Installation

Get started in seconds

1

Add the Marketplace

/plugin marketplace add yajinn/claude-tool-tracker
2

Install the Plugin

/plugin install claude-tool-tracker@yajinn-plugins
3

Restart & Use

Restart Claude Code. Every tool call will now be tracked!

/tool-stats

Commands

Control your tracking experience

Command Description
/tool-stats Show current session statistics
/tool-stats --all Show all-time statistics
/tool-theme colorful Use colorful boxed theme
/tool-theme minimal Use minimal single-line theme
/tool-theme emoji Use emoji prefix theme
/tool-config View and modify configuration
